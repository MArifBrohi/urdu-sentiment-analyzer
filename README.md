# 🌍 Roman Urdu Sentiment Analyzer

A Machine Learning web app that analyzes sentiment of Roman Urdu & English text.

## 🔗 Live Demo
[Click here to try the app](https://urdu-sentiment-analyzer-movie-based.streamlit.app)

## 📊 Model Performance
- Algorithm: TF-IDF + Logistic Regression
- Accuracy: 80.49%
- Training data: 11,018 Roman Urdu reviews

## 🛠️ Tech Stack
- Python, Scikit-learn
- TF-IDF Vectorizer
- Streamlit
- Pandas, NumPy
